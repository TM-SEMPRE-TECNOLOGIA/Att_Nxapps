# 🚀 PREVC — Inteligência do Projeto Att_Nxapps

Este diretório guarda a **memória organizada do projeto** seguindo a metodologia PREVC da AI Coders Academy.

Cada sprint cria documentos nas 5 pastas abaixo, garantindo rastreabilidade total do que foi planejado, revisado, executado, testado e confirmado.

## Estrutura

| Pasta | Fase | O que contém |
|---|---|---|
| `planning/` | 🎯 Planning | Planos de sprint, escopo, estimativas |
| `review/` | 🔍 Review | Auditorias CrewAI pré-execução |
| `execution/` | ⚡ Execution | Changelogs e logs de implementação |
| `validation/` | ✅ Validation | Resultados de testes e verificações |
| `confirmation/` | 📋 Confirmation | Resumos executivos, links para commits |

## Convenção de Nomenclatura

```
sprint_01_plan.md
sprint_01_review.md
sprint_01_changelog.md
sprint_01_tests.md
sprint_01_confirmation.md
```
